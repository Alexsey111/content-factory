# AI Content Factory

Мини-система для генерации и автоматической публикации контента в Telegram.

---

## 🚀 Что делает система

1. Генерирует посты по темам
2. Создаёт изображения
3. Складывает всё в очередь
4. Публикует в Telegram по расписанию

---

## 🧱 Архитектура

topics → генерация → очередь → воркер → Telegram

---

## 📦 Установка

```bash
pip install -r requirements.txt
```

## ⚙️ Как работать (ВАЖНО)

### 🔹 Режим 1: Генерация контента

```bash
python main_generate.py
```

Что происходит:

* создаются посты
* создаются изображения
* всё добавляется в очередь (`storage/queue.json`)

👉 Этот шаг запускается **1 раз**, когда нужно создать контент

---

### 🔹 Режим 2: Публикация (воркер)

```bash
python main_worker.py
```

Что делает воркер:

* работает бесконечно
* каждые 30 секунд проверяет очередь
* публикует посты, у которых подошло время

👉 Этот процесс должен работать **постоянно**

---

## ❗ Важно понимать

Ты НЕ запускаешь один файл “и всё само”.

Правильная модель:

| Процесс          | Назначение         |
| ---------------- | ------------------ |
| main_generate.py | генерирует контент |
| main_worker.py   | публикует          |

---

## 🧪 Типовой сценарий

1. Добавил темы
2. Запустил генерацию:

```bash
python main_generate.py
```

3. Запустил воркер:

```bash
python main_worker.py
```

👉 дальше всё идёт автоматически

---

## 🔄 Как работать ежедневно

### Добавить новые посты:

* редактируешь `topics.txt`
* снова запускаешь `main_generate.py`

### Публикация:

* воркер уже работает → ничего делать не нужно
