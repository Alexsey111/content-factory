# AI Content Factory

Генерация Telegram-постов + изображений из CSV таблицы курсов.

## 🚀 Возможности

- Генерация текста постов
- Генерация изображений (1080x1080)
- Обработка CSV
- Масштабируемый pipeline

python -m venv .venv
source .venv/bin/activate  # Linux / Mac
.venv\Scripts\activate     # Windows
